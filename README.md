**DebateGPT**
AI-Based Real-Time Debate Analyzer & Counter-Argument Chatbot

DebateGPT is an AI-powered system that helps users practice debating, analyze speeches, and receive real-time feedback on clarity, argument quality, grammar, logic, and tone.
It integrates Speech-to-Text, NLP argument mining, sentiment analysis, and grammar evaluation into a single, easy-to-use platform.

This project has two major components:
Features
1. Debate Practice Chatbot
- Engage in a multi-turn debate with an AI chatbot.
- Provides claims, counter-arguments, rebuttals, and topic-based responses.
- Helps users practice structured debating.

2. Real-Time Debate Analyzer (Audio → Text → Feedback)
- Live speech transcription using Vosk STT.
- Real-time noise reduction for cleaner audio.
- NLP-based analysis: Argument detection (Claim, Evidence, Counter, Rebuttal)
- Sentiment & tone analysis

Grammar & fluency evaluation
Generates a structured performance report.
